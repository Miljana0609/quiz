# 🎯 Personal Quiz App

This is a simple and interactive quiz about me, built while revisiting the basics of frontend development.

After focusing on backend development, I recently started working with frontend and wanted to refresh my knowledge of **HTML, CSS, and JavaScript** before diving into frameworks.

## 💡 About the project

The quiz is:

* Simple and fun to interact with
* Built using vanilla JavaScript
* Enhanced with GIFs from Giphy for a more engaging experience
* Designed so you can’t really fail 😉

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 🎯 Purpose

This project was created to:

* Revisit frontend fundamentals
* Practice DOM manipulation
* Build something small but interactive
* Get comfortable deploying with GitHub Pages
